BOOTSTRAP 5 PRO Y UTILIDADES AVANZADAS

Trucos avanzados de la grilla, espaciado responsivo y personalización.

Puntos de Ruptura (Breakpoints) Responsivos

Bootstrap define 6 puntos de ruptura. Entenderlos es crucial para aplicar clases utilitarias de forma precisa en diferentes tamaños de pantalla.

Breakpoints de Bootstrap 5:
Clase Prefijo Tamaño de Pantalla
<576px`xs` (ninguno)Extra Small (Móviles)
$\ge$ 576px`sm`Small (Móviles grandes)
$\ge$ 768px`md`Medium (Tablets)
$\ge$ 992px`lg`Large (Desktops)
$\ge$ 1200px`xl`Extra Large (Desktops grandes)
$\ge$ 1400px`xxl`XXL (Monitores ultra anchos)
Ejemplo de Uso Avanzado:

Muestra un elemento en escritorio (`lg`) y lo oculta en móvil (`md` y menores).

<!-- Visible en lg y xl, oculto en md, sm y xs -->
<div class="d-none d-lg-block">Mi Sidebar de Escritorio</div>

<!-- Oculto en md y mayores, visible en sm y xs -->
<div class="d-md-none">Mi Menú de Navegación Móvil</div>
Columnas de Auto-Layout (`col`)

Utiliza la clase `col` sin número para que las columnas se dividan el espacio disponible de manera equitativa. Útil cuando no sabes cuántas columnas tendrás.

Ejemplo Básico (Distribución Equitativa):
<div class="row">
  <div class="col">Columna 1</div>
  <div class="col">Columna 2</div>
  <div class="col">Columna 3</div>
</div>
<!-- Las tres columnas ocuparán el 33.3% del ancho -->
Ejemplo Avanzado (Contenido Variable):

Asigna un tamaño fijo a una columna y deja que las demás se ajusten.

<div class="row">
  <div class="col">Contenido Flexible</div>
  <div class="col-6">Contenido Fijo (50% de ancho)</div>
  <div class="col">Contenido Flexible</div>
</div>
<!-- Las columnas flexibles dividirán el 50% restante (25% cada una) -->
Utilidades de Margen y Padding Responsivas

Las utilidades de espaciado te permiten controlar el margen (`m`) y el padding (`p`) de manera muy granular, combinando lados, tamaños y puntos de ruptura.

Sintaxis General:
{propiedad}{lado}-{breakpoint}-{tamaño}
Ejemplos Listos:
<!-- Margen superior de 3 en todos los tamaños (m = margin, t = top, 3 = tamaño) -->
<div class="mt-3">...</div>

<!-- Padding horizontal de 5 en escritorio (p = padding, x = eje horizontal, lg = large) -->
<div class="px-lg-5">...</div>

<!-- Eliminar margen inferior en tablets y superior en móviles -->
<div class="mb-md-0 pt-sm-0">...</div>

<!-- Centrar un bloque: Margen horizontal automático -->
<div class="mx-auto" style="width: 300px;">...</div>
Tabla de Referencia Rápida:
  • **Propiedades:** `m` (margin), `p` (padding).
  • **Lados:** `t` (top), `b` (bottom), `s` (start/left), `e` (end/right), `x` (horizontal), `y` (vertical).
  • **Tamaños:** 0 a 5 (escalado predefinido), o `auto`.
Fijación de Elementos (Sticky / Fixed)

Controla si un elemento debe permanecer visible en la pantalla incluso al hacer scroll. Muy útil para headers y sidebars.

Ejemplo: Header Fijo y Sidebar Flotante
<!-- Fija el elemento en la parte superior de la pantalla -->
<header class="fixed-top bg-dark p-3">...</header>

<!-- Fija el elemento en la parte superior del padre, solo se activa al hacer scroll -->
<div class="sticky-top bg-light p-2">...</div>

<!-- Fija el elemento en la esquina inferior izquierda -->
<div class="fixed-bottom start-0">...</div>
Clases Clave:
  • **`fixed-top`**: Posicionamiento fijo en la parte superior.
  • **`fixed-bottom`**: Posicionamiento fijo en la parte inferior.
  • **`sticky-top`**: El elemento se comporta como `relative` hasta que alcanza el borde superior de la ventana gráfica, donde se vuelve `fixed`.
  • **Nota:** Si usas `fixed-top` o `fixed-bottom`, añade padding al `` para evitar que el contenido quede oculto debajo.
Personalización con Variables SASS (Avanzado)

Para personalizaciones profundas, la mejor práctica es modificar las variables de Bootstrap usando SASS, como cambiar los colores primarios y secundarios o los radios de borde.

Ejemplo de SASS (Archivo `_custom.scss`):
// 1. Sobrescribir variables de color predeterminadas
$primary: #00bcd4; // Mi nuevo color primario (Cian)
$success: #8bc34a; // Mi nuevo color de éxito

// 2. Sobrescribir variables de espaciado o tipografía
$spacer: 1.5rem; // Cambia el valor base de '1' a 1.5rem
$border-radius: 0.5rem; // Aumenta el radio de los bordes

// 3. Importar Bootstrap para que compile con tus nuevas variables
@import "bootstrap/scss/bootstrap";
Implementación:
  • **Requerimiento:** Debes tener SASS instalado y configurar un proceso de compilación.
  • **Ventaja:** Este método es mucho más limpio que sobrescribir clases en CSS puro y permite personalizar toda la apariencia de Bootstrap.
Aplicar Sombras Rápidas (`shadow`)

Usa las clases de sombra para dar profundidad a elementos como tarjetas o modales, sin escribir código CSS de caja-sombra (`box-shadow`).

Clases de Sombra:
<!-- Sin sombra -->
<div class="shadow-none">...</div>

<!-- Sombra pequeña -->
<div class="shadow-sm">...</div>

<!-- Sombra predeterminada -->
<div class="shadow">...</div>

<!-- Sombra grande -->
<div class="shadow-lg">...</div>
Ejemplo:
  • Combínalo con tarjetas para un efecto 3D: `<div class="card shadow-lg p-3">...</div>`
