CSS MAESTRO Y TRUCOS MODERNOS

Aprende Flexbox, Grid, variables CSS y funciones de diseño fluido.

Variables CSS (Propiedades Personalizadas)

Permiten definir valores que se pueden reutilizar en todo el CSS. Son cruciales para mantener la consistencia y cambiar temas fácilmente.

Declaración y Uso (en tu `styles.css`):
:root {
  /* Declaración global en la raíz */
  --color-principal: #007bff;
  --espacio-lg: 3rem;
}

.elemento {
  /* Uso de la variable */
  background-color: var(--color-principal);
  padding: var(--espacio-lg);
}
Uso:
  • **Declaración:** Siempre con doble guion inicial (`--nombre`).
  • **Acceso:** Con la función `var(--nombre)`.
  • **Ventaja:** Permiten cambiar el valor con JavaScript o Media Queries.
Flexbox: Alineación en una Dimensión

Ideal para alinear ítems a lo largo de un eje (fila o columna) y distribuir espacio de manera eficiente, especialmente en componentes pequeños.

Ejemplo: Centrar Contenido Vertical y Horizontalmente
.contenedor-centrado {
  display: flex;
  justify-content: center; /* Centrado horizontal */
  align-items: center;    /* Centrado vertical */
  height: 100vh;
}
Propiedades Clave:
  • **`display: flex;`**: Convierte el elemento en un contenedor flexible.
  • **`justify-content`**: Controla la distribución a lo largo del eje principal.
  • **`align-items`**: Controla la alineación a lo largo del eje secundario.
Grid: Maquetación en Dos Dimensiones

El sistema definitivo para crear estructuras complejas de dos dimensiones (filas y columnas), perfecto para el layout principal de una página web.

Ejemplo: Sidebar Fijo y Contenido Flexible
.layout-pagina {
  display: grid;
  /* Define 2 columnas: 250px fijo y el resto del espacio (1fr) */
  grid-template-columns: 250px 1fr;
  /* Nombra las áreas para mejor claridad */
  grid-template-areas: "sidebar contenido";
}

.menu-lateral {
  grid-area: sidebar;
}
.contenido-principal {
  grid-area: contenido;
}
Unidades Clave:
  • **`display: grid;`**: Convierte el elemento en un contenedor de cuadrícula.
  • **`grid-template-columns` / `rows`**: Define la estructura de la cuadrícula.
  • **`fr` (Fractional Unit)**: Asigna una porción fraccional del espacio disponible, ideal para diseño responsivo.
Diseño Fluid-Type con `clamp()`

Asegura que el tamaño de la fuente (u otras propiedades) se mantenga entre un mínimo y un máximo, escalando suavemente en función del tamaño de la ventana (Viewport).

Uso:
h1 {
  /* Tres valores: Mínimo, Preferido (fluido), Máximo */
  /* El tamaño de la fuente irá entre 2rem y 5rem. */
  font-size: clamp(2rem, 10vw + 1rem, 5rem);
}

.caja {
  /* También útil para padding o márgenes */
  padding: clamp(1rem, 5vw, 3rem);
}
Sintaxis:
  • **`clamp(Mínimo, Valor_Preferido, Máximo)`**
  • **`Valor_Preferido`**: Usa unidades relativas como `vw` (Viewport Width) para el escalado.
Selector de Negación `:not()`

Aplica estilos a todos los elementos excepto al que se especifica. Ideal para reglas de espaciado o bordes, evitando tener que anular estilos en el último elemento.

Ejemplo: Separador entre elementos (menos el último)
/* Aplica un margin-right a todos los .item */
/* EXCEPTO al último hijo */
.lista-items .item:not(:last-child) {
  margin-right: 1.5rem;
}

/* Opcional: A todos los elementos .post EXCEPTO a los .destacado */
.post:not(.destacado) {
  opacity: 0.8;
}
Uso:
  • **`:not(selector)`**: Invierte la selección. Funciona con clases, IDs y otros selectores pseudo.
  • **Mejor Práctica:** Ayuda a limpiar el CSS eliminando reglas de anulación como `*:last-child { margin-bottom: 0; }`.
Scrollbars Personalizados (Webkit)

Si bien no es estándar, la personalización de la barra de desplazamiento mejora la estética en navegadores basados en Webkit (Chrome, Edge, Safari).

Código para `styles.css`:
/* 1. Define el ancho de la barra */
::-webkit-scrollbar {
  width: 10px; 
}

/* 2. El 'track' es el camino por donde se mueve la barra */
::-webkit-scrollbar-track {
  background: var(--color-dark-indigo);
  border-radius: 10px;
}

/* 3. El 'thumb' es la parte que arrastras */
::-webkit-scrollbar-thumb {
  background: var(--color-cyan);
  border-radius: 10px;
}

/* Opcional: Estado hover */
::-webkit-scrollbar-thumb:hover {
  background: var(--color-pink);
}
Nota Importante:
  • **Compatibilidad:** Funciona en Chrome, Edge, Safari y Opera. Para Firefox, se usan las propiedades `scrollbar-width` y `scrollbar-color`.
Transiciones Suaves en `:hover`

Asegúrate de que los cambios de estilo al interactuar (como pasar el ratón) sean suaves y agradables, no abruptos.

Ejemplo: Efecto Glow en Botones
.btn-glow {
  /* La clave: Aplica la transición al estado normal */
  transition: all 0.3s ease-in-out;
  border: 2px solid var(--color-cyan);
  box-shadow: 0 0 5px var(--color-cyan);
}

.btn-glow:hover {
  /* Los estilos de cambio */
  background-color: var(--color-cyan);
  color: var(--color-dark);
  box-shadow: 0 0 10px var(--color-cyan), 0 0 20px var(--color-cyan);
  transform: scale(1.05);
}
Regla de Oro:
  • **Ubicación de `transition`**: Siempre en el selector base (`.btn-glow`), no en el `:hover`.
